# Interactive Tic-Tac-Toe Game 🎮

A sleek, responsive, and fully interactive Tic-Tac-Toe game built from scratch using HTML5, CSS3, and Vanilla JavaScript. This project showcases dynamic DOM manipulation, state management, and win-condition validation algorithms.

## ✨ Features
* **Smooth Turn Management:** Dynamically alternates between 'X' and 'O' players with clear visual turn indicators.
* **Win & Draw Detection:** Implements an algorithmic check against all 8 possible winning combinations after every move, triggering immediate win or draw screens.
* **Interactive UI:** Hover highlights on empty cells show the current player's token translucent preview to enhance user experience.
* **Instant Reset Functionality:** A "Reset/New Game" button clears the board array and resets the game state without needing a page reload.
* **Fully Responsive:** Scalable grid system that adapts effortlessly to desktop, tablet, and mobile screens.

---

## 🛠️ Tech Stack
* **HTML5:** Structured the 3x3 game board grid and UI overlays.
* **CSS3:** Custom grid layout, flexbox alignment, glowing transitions, and responsive scaling.
* **JavaScript (ES6+):** Event listeners, multi-dimensional array tracking for board state, and conditional logic for win validation.

---

## 🧠 Technical Learnings
Developing this game helped me sharpen several key programming fundamentals:
1.  **State Management:** Maintained a clean virtual representation of the 3x3 board in JavaScript memory to mirror and validate the UI state.
2.  **DOM Manipulation:** Leveraged event delegation to capture user clicks dynamically on grid cells and update their classes and inner text in real time.
3.  **Algorithmic Thinking:** Translated the physical rules of Tic-Tac-Toe into code by mapping out arrays of index combinations to detect winning patterns.
